# RETROBOX

RETROBOX es una tienda web ficticia dedicada a la compra y alquiler de videojuegos y consolas retro. El proyecto está desarrollado como una SPA (Single Page Application) sencilla, usando HTML, CSS y JavaScript puro, con un diseño visual inspirado en la estética retro y efectos glitch.

## Estructura del Proyecto

- **index.html**: Página principal con información general y navegación.
- **comprar1.html, comprar2.html, comprar3.html**: Páginas para explorar y comprar videojuegos retro.
- **alquilar1.html, alquilar2.html, alquilar3.html**: Páginas para alquilar consolas retro.
- **detallealquilar.html, detallecomprar.html**: Detalles de productos para alquiler o compra.
- **contacto.html**: Formulario de contacto.
- **notfound.html**: Página de error o enlaces ficticios.
- **css/style.css**: Estilos personalizados con paleta retro y efectos glitch.
- **js/script.js**: Lógica de interacción, menú, radio online, scroll, etc.
- **audio/**: Carpeta para archivos de audio (ej: música de fondo).
- **img/**: Carpeta para imágenes y recursos gráficos.

## Características

- Navegación entre páginas de compra, alquiler y contacto.
- Reproductor de radio integrado con control de volumen.
- Diseño responsive y visual retro (paleta púrpura, tipografías pixeladas).
- Efectos visuales glitch y animaciones suaves.
- Footer con enlaces rápidos, contacto y redes sociales ficticias.

## Instalación y Uso

1. Descarga o clona este repositorio.
2. Abre `index.html` en tu navegador web.
3. Navega por las diferentes secciones usando el menú.

No requiere backend ni instalación de dependencias.

## Créditos

- Tipografías: Google Fonts (Press Start 2P, VT323)
- Iconos y recursos visuales: libres o de uso educativo.
- Música: Añade tu propio archivo MP3 en la carpeta `audio/` si lo deseas.
